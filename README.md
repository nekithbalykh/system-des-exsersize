# Library Check-In / Check-Out System — System Design

A system design exercise covering a library check-in/check-out platform that allows users and admins to:

- Check out a book (remote reservation + in-person pickup confirmed by admin)
- Return a book (in person; return confirmed by admin)
- Add a book to the system (admin only)

Full system design: [DESIGN.md](./DESIGN.md)
